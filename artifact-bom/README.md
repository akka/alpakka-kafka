# artifact-bom

The Maven dependency descriptors in the `artifact-bom` directory 
support our security tooling. These are generated by explicit activation of 
the [sbt-artifact-bom plugin](https://github.com/akka/sbt-artifact-bom/) on the 
sbt projects.

File generation runs on `compile`. It should be enough to add those generated files
to the PR when changing the dependencies.
